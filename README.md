# Quiz App

## Setup Instructions

1.  **Unzip the build**: Extract the contents of `quiz_app_build.zip`.
2.  **Install Dependencies**: Open a terminal in the folder and run:
    ```bash
    npm install
    ```
3.  **Start the Server**:
    ```bash
    node server.js
    ```
4.  **Access the App**: Open your browser and go to `http://localhost:3000`.

## Features
- **Java Quiz**: Test your Java knowledge.
- **Leaderboard**: Real-time global leaderboard with rank tracking.
- **Privacy**: Usernames are masked for privacy.
